# 🩺 Diabetes Prediction System

A Machine Learning powered web application that predicts whether a person is likely to have diabetes based on medical parameters.

Built using:
- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly

---

# 🚀 Features

- Real-time Diabetes Prediction
- Interactive Dashboard
- Data Analytics & Visualizations
- Feature Importance Analysis
- Probability Score Prediction
- Responsive UI

---

# 🧠 Machine Learning Model

Model Used:
- Random Forest Classifier

Preprocessing:
- Feature Scaling using StandardScaler

Evaluation Metrics:
- Accuracy Score
- Confusion Matrix
- Classification Report

# ▶️ Running Locally

## Install Dependencies

```bash

```

## Run Application

```bash
streamlit run app/app.py
```
